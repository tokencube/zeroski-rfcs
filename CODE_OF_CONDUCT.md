# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1][covenant] as its
code of conduct. By participating in this repository — opening an RFC,
commenting on a PR, filing an issue — you agree to abide by its terms.

[covenant]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Reporting

Report conduct concerns privately to **`conduct@zero.ski`**. Reports are read
by the project maintainer ([@zhanjun](https://github.com/xczhanjun)) and are
treated as confidential.

Include, to the extent you are comfortable:

- What happened, where, and when.
- Who was involved.
- Any relevant links (PRs, issues, messages).
- Whether you want to be contacted back.

The maintainer will acknowledge the report within 72 hours and describe the
next steps. Enforcement follows the Community Impact Guidelines described in
the Contributor Covenant (correction → warning → temporary ban → permanent
ban), scaled to the severity and pattern of the behaviour.

## Scope

This code of conduct applies to all project spaces — this repository, its
issues and PRs, and any official channel (email, chat) that identifies with
the project.
